# 《金匮要略》底本候选盘点

盘点日期：2026-07-16

## 结论

首批不宜直接采用来源不明的网络纯文本。建议先建立“两主本一参校本”：

1. **元邓珍本系统**：以《四部丛刊初编》影印《新编金匮要略方论》为首个主本。
2. **明赵开美本系统**：以《仲景全书》所收《金匮要略方论》为第二主本。
3. **后世整理或注本**：只作为参校层，不与主本文字混合。

## 已核候选

### 四部丛刊本《新编金匮要略方论》

- 系统：元邓珍本系统。
- 载体：影印 PDF，可逐页核对。
- 候选来源：
  - 中国哲学书电子化计划：https://ctext.org/jinkui-yaolue/zh
  - Wikimedia Commons：https://commons.wikimedia.org/wiki/Category:金匱要略方論
- 当前判断：适合作为首个导入底本。
- 风险：OCR 文本错误较多，必须以影像为准；需核对册次是否完整。

### 明赵开美本《金匮要略方论》

- 系统：赵开美《仲景全书》本。
- 载体：已有影印出版和版本研究记录。
- 书目研究参考：https://square.umin.ac.jp/mayanagi/students/05/noguchi/chap4.html
- 当前判断：适合作为第二主本，与现有赵开美系统《伤寒论》形成一致的版本坐标。
- 风险：尚未确认可直接下载、可重复利用的完整公共影像。

### 中国哲学书电子化计划 OCR 本

- 地址：https://ctext.org/wiki.pl?if=zh&res=416678
- 当前判断：只能作为 OCR 候选和检索辅助，不可直接当作校定原文。
- 原因：平台明确提示文本由影像自动 OCR 生成，抽样可见大量识别错误。

## 推荐实施顺序

1. 下载并登记四部丛刊本各册影像。
2. 检查缺页、重页、清晰度和版权信息。
3. OCR 后按篇章切分，生成 `imported` 状态的文本单元。
4. 人工抽检篇名、方名、剂量和方后煎服法。
5. 获得赵开美本影像后建立跨版本对应关系。
