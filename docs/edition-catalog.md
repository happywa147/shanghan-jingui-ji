# 《伤寒论》《金匮要略》版本总目录与首批主本

盘点日期：2026-07-18

## 口径

本目录的“全版本”范围是当前能在中文维基文库与 Wikimedia Commons 明确识别、追溯的版本组，不把同一版本的上下册、卷册或重复扫描计为多个版本。原典底本、后世注本、现代排印和维基校录分层登记。

机器目录见 `data/sources/edition-catalog.json`；执行 `npm run verify:edition-catalog` 检查唯一ID、必填证据和每书3个首批主本。

## 去重与分层结果

- Commons《伤寒论》分类所称94项是文件数，包含分册、重复扫描及注本，不能作为94个版本宣传。
- 《金匮》可识别到四部丛刊、赵开美、吴迁抄本、四部备要、民国排印等底本或排印本；《心典》《论注》《直解》《集解》《编注》《浅注》《浅注补正》《辑义》《高注》《五十家注》等已分别立项，归参校层。
- 《伤寒》除首批三本外，长沙古本属于从《义疏》析出的派生文本；成无己注本及《三注》《集注》归注本或文本见证层。
