# v0.3 评审整改台账

## 已完成

- 将误导性的“原文”语义改为“来源整理文本”，增加卷次、来源校勘、康平主文、结构层级和扫描定位槽位。
- 将 30 组被重复强制为一对一的康平对应归并为多对一；对应模型新增无对应、未判定、增文和缺文类型。
- 审核导出升级为 v2，记录审核者标识、角色、输入修订号和身份未验证声明，支持初审、复审与裁决的独立导出。
- 修复 HTML 注入、来源路径逃逸；下载增加 HTTPS 主机允许列表、60 秒超时、流式写入和登记尺寸上限。
- 方剂页显示煎服法与数据复核状态，增加有毒药物、古剂量不可照用和非医疗建议警示。
- 修复下一条未审跳出当前筛选、本地存储失败无提示、移动端换行、表单标签和焦点可见性问题。
- 引入 JSON Schema 2020-12 真实运行时校验、Node.js 22 CI、依赖锁定、漏洞审计和 GitHub Pages 发布工作流。

## 公开质量边界

- 779 组版本对应和 161 组同名方对照均为机器候选，人工复核数为 0。
- 《金匮要略》影像 2 册共 264 页已校验哈希与页数；OCR 和逐页人工核对未完成，本版不发布《金匮》校定文本。
- 发布定位是“文献校勘候选”，不是校定本、临床决策系统或处方工具。
