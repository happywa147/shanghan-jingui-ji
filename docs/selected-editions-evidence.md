## 桂林古本争议证据包

当前只能将其列为“有转录、来源阻断”：

1. 维基转录序言称其传承为张绍祖—左盛德—罗哲初—罗继寿，但这属于该版本自身的传承陈述，不能替代原抄本或独立馆藏证据。
2. 1986年中医整合研究小组序明确说明整理时未取得原抄本比对，并承认错误难免。
3. 当前转录加入了标点、条号、索引并进行繁简和版式处理，因此是二次整理层，不是原抄本影像层。
4. 原作可按公版处理，维基页面贡献按CC BY-SA 4.0署名共享；“可合法转录”不等于“版本传承已证实”。

结论：保留为争议参校本；在获得可逐页定位的原抄本或早期刊本影像前，不进入黄金文本，不以其补齐其他版本缺文。
