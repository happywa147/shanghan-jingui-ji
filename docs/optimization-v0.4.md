# v0.4 黄金样本与差异分类

## 本轮产物

- 从 779 组条文对应中确定性抽取 50 条黄金样本候选，覆盖多对一合并、低信度、高差异率、完全相同和版本平衡。
- 候选默认状态是 `candidate`，不得当作已验证的评测真值。
- 建立独立初审、独立复审、第三人裁决门禁；系统拒绝同人自审自复审或参与裁决。
- 将方剂差异拆分为药味数量、药味/次序、剂量、炮制、煎服内容和煎服句读六类。

## 结果

- 161 组同名方对照中，159 组存在原始字段差异。
- 其中 11 组只有煎服句读或编辑层差异，非句读实质差异候选为 148 组。
- 上述“实质”仅指数据字段层面，不等于已确认的医理影响。
