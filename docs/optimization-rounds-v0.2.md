# v0.2 十轮优化记录

1. 增加数据质量统计报告，量化低信度、缺篇名和无组成方剂。
2. 建立以宋本为轴的同名方剂差异引擎。
3. 生成可检索、可筛选的方剂对照报告。
4. 校勘台增加审核进度和“下一条未审核”。
5. 移除导入时间漂移，同一来源可生成字节一致的产物。
6. 强化来源许可、日期、HTTPS、页数、尺寸、哈希与重复路径检查。
7. 增加 `npm run status` 项目状态总览。
8. 方剂比较回归测试加入剂量、炮制和药味顺序边界。
9. 校勘报告增加内容安全策略、可访问名称和损坏缓存恢复。
10. 将全部新增步骤纳入 `npm run check`，形成 v0.2 验收基线。

## v0.2 验收条件

- 自动测试全部通过。
- 三版导入与来源影像校验通过。
- 条文与方剂差异产物可重复生成。
- 两套静态报告条目数与数据一致。
- Git 工作树干净并创建版本标记。
