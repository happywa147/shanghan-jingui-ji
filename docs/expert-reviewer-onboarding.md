# 真人专家审核池：五席配置与入池规范

五席分别覆盖《伤寒论》版本学、《金匮要略》版本学、中医临床与方药安全、古籍整理与文字校勘、独立裁决与质量验收。

五席是专家池容量，不表示每条需要五人审核。每条由两名不同真人独立审核；意见不一致时，由第三名未参与前两审的专家裁决。AI agent 只能产生预审建议，不能登记为真人审核者。

入池需提供可公开核验的审核标识，及学历、职称、执业资格、古籍整理成果或研究经历至少一项；同时登记可承担角色、利益冲突、资格核验日期和公开证据。项目不保存身份证号、联系方式等非必要敏感信息。

单条审核必须查看底本文字、影像页、卷篇条定位和机器差异，并填写决定、关系类型、证据引用、说明和审核时间。缺字、正文/注文边界、疑似OCR字及高风险方药不得仅凭机器相似度通过。
